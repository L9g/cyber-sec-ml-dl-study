# 交易图 GNN:图到底有没有用?——三层对照拆开「NN vs 树」与「图 vs 无图」

> 一步步拆解「在交易图上跑 GraphSAGE,它比表格模型强吗?——不强;但只报『GNN < GBDT』会把账算错,
> 三层对照显示图结构其实加了一点点,大头缺口是『NN 打不过树』而非『图打不过无图』」。
> 数据来源:`notebooks/07_tx_graph_gnn.py` + `results/experiments.csv`(temporal split，CPU 训练，无需 GPU)。

## 0. 为什么要三层对照

项目四论点:「图 > 表格」**不自动成立**(Weber 2019 自报 RF > 普通 GCN)。但「GraphSAGE 0.66 < LightGBM 0.81」
这句话有**两个混淆**搅在一起:

1. **NN vs 树**:GraphSAGE 是神经网络,LightGBM 是梯度提升树。树在工程好的表格特征上常更强。
2. **图 vs 无图**:消息传递到底加没加信号。

要干净回答「图有没有用」,必须加一个**中间控制项**:MLP(同特征、同 NN 训练、**但没有边**)。
则 **SAGE − MLP = 纯消息传递增益**(唯一差别是图),**MLP − LightGBM = NN vs 树**的代价。

## 1. 结果:三层对照

同 temporal split(前 34 训 / 后 15 测)、同 182 维交易特征、测试集 illicit base rate 6.5%:

| 模型 | 特征 | 图消息传递 | PR-AUC |
|---|---|---|---|
| LightGBM | 182 维 | ✗ | **0.813** |
| MLP | 同上 | ✗ | **0.624** |
| GraphSAGE | 同上 | ✓ | **0.660** |

两个关键差值:

- **纯消息传递增益 = SAGE − MLP = +0.036**(图**确有**微弱信号,非零)。
- **NN vs 树 = MLP − LightGBM = −0.189**(大头缺口在这)。
- 合计 **GNN vs GBDT = SAGE − LightGBM = −0.153**。

## 2. 读法:别把账算错

- **只报「GNN 0.66 < GBDT 0.81」会误导**——会让人以为「图没用/图有害」。真相是:**图结构加了 +0.036**,
  只是被「NN 在这些工程特征上本就打不过树」的 −0.19 淹没了。
- **图为什么只加一点点**:交易图**按时间步完全断开**(复验:所有边两端 Δ(Time step)=0)→ 消息传递**只在同时间步内**发生。
  同期邻居往往是同一洗钱链上的相邻交易,而**特征里已编码了大部分同期信息**(btc 量/度/聚合统计),
  再做一跳同期聚合边际自然小。
- **顺带一个结构红利**:因为边不跨期,temporal split 在**图层面天然 inductive**——测试子图与训练子图无共享边,
  测试节点的邻居全是测试期节点,无结构泄漏。

## 3. 对旗舰问句 + 「为什么用图」三段论的收口

- **「图 > 表格」不自动成立**,在本数据实证成立(呼应 Weber 2019 RF>GCN、nb03 裸拓扑近随机)——
  判别信号**主要在工程特征**,图结构只贡献很小的同期增益、且换不回 NN 相对 GBDT 的劣势。
- **图的"大"增益只可能来自**:(a) **跨期时序**(EvolveGCN 类,在时间上传播)——但**交易图跨期断开**,
  故这条要到**地址图**(AddrAddr 有跨期边)才谈得上;(b) 结构感知/混合范式。
- 地址图是 **2.87M 边**的更大图,按算力约束([[project-compute-no-gpu-cloud-fallback]]:本机无 GPU)
  留云主机或邻居采样。**当前 CPU 交易图 GNN 闭环到此**。
- ⚠️ **retrospective 未解**:即便换 GNN,标签仍是全局/事后裁定(provenance 主线不因换模型而变)。

## 4. 这一段在项目里的位置

前面几段拆的都是**标签/评估口径**(切分、聚合、unknown、投影);本段拆的是**模型范式**:

- **给"为什么用图"一个诚实的实证答案**:在这个同期断开、特征工程充分的交易图上,图不自动赢;
  想让图赢,得去有跨期结构的地址图,且要过强 GBDT 这关。
- 方法学复用价值:**三层对照(GBDT / NN-no-graph / GNN)是"图有没有用"的正确问法**——
  不加 MLP 控制项,就会把「NN vs 树」的账错记到「图 vs 无图」头上。

一句话记忆点:**GraphSAGE 输给 LightGBM,但输的是"神经网络打不过树",不是"图打不过无图";
图其实加了 +0.036——只是在一个按时间步断开的交易图上,这就是它能加的全部。**
