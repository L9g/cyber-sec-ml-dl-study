# 专题:选择性标注、reject inference 与 IPW —— 为什么"把未审当负样本"是头号错误

> 类型:**原理讲解 / 教学文档**(2026-07-04)。配套实验:`docs/fraud-aml-label-pipeline-experiment-plan.md` §6(Stage B)、
> 骨架代码 `04-aml-gnn/notebooks/13_selective_labeling_reject_inference.py`。
> 目的:用通俗语言 + 数学 + 代码,讲清 fraud/AML 标签管线里最常见、也最致命的一种偏差,以及它的缓解与**边界**。
> 一句话:**真实建模的头号错误 = 把"没人审过 / 被放行"的交易当干净负样本;主缓解不是花哨算法,是纪律
> ——只用被真正审核 / 裁定过的标签训练(reject inference);IPW 只是小精修,面试要主动讲它纠不动的地方。**

---

## 0. 为什么这值得单独写一篇

Kaggle 上的 fraud 数据集标签是干净、完整的,所以"GBDT + 调参"就能拿高分,像个玩具。真实世界不是这样:
**标签不是数据自带的真相,而是一条会出错、会被旧策略偏置的生产管线的输出。** 这篇讲的就是这条管线里
第一号病理——**选择性标注(selective labeling)**:你只对被审查过的案子知道真相,其余的你**假装**知道(当成负样本)。
这个"假装"就是大多数 fraud/AML 模型悄悄崩掉的地方。

---

## 1. 一个比喻(先装进脑子)

医院只给"看起来像病人"的人做检测(风控只审**金额高**的交易)。没做检测的人,你在表格里默认写"健康"(未审→负样本)。
问题是:**有些没被检测的人其实有病**(低金额的洗钱没人查)。于是你的"健康"堆里混进了真病人,模型学到的
"什么样算病"就是歪的。

- **头号错误** = 把"没检测的"当"健康的"。
- **主缓解(reject inference)** = 训练时**只用真正做过检测、有确诊结果的人**,绝不把"没检测"硬写成"健康"。
- **IPW** = 因为被检测的人里"像病人的"过多,把他们**按稀有度加权**,让训练集看起来像整个人群。对灵活模型来说这步经常可有可无。

---

## 2. 记号:标签到底怎么来的

| 符号 | 含义 |
|---|---|
| `x` | 一笔交易 / 一个账户的特征(金额、对手方…) |
| `y ∈ {0,1}` | **真相**(是否洗钱)。现实里看不全 |
| `S ∈ {0,1}` | 这笔**有没有被审查 / 裁定**。**只有 `S=1` 才拿得到 `y`** |
| `e(x) = P(S=1 \| x)` | **倾向分(propensity)**:给定特征被审查的概率。旧规则"审高金额"⇒ `e(x)` 随金额升高 |
| `η(x) = P(y=1 \| x)` | 我们真正想学的:全人群上的风险 |

目标:在**整个人群**上学好 `η(x)`(或一个好排序)。麻烦在于手上的标签只覆盖 `S=1` 那部分。

---

## 3. 头号错误:assume-negative 为什么在数学上必然偏

把"未审"填成 0,等于用了一个**伪标签**:

```
ỹ = y      if S = 1
ỹ = 0      if S = 0
```

看它的条件期望,并用 **MAR 假设**(给定 `x`,审不审与真相无关:`P(y|x,S)=P(y|x)`):

```
E[ỹ | x] = P(S=1|x)·E[y | x,S=1] + P(S=0|x)·0
         = e(x) · η(x)
```

**关键**:你让模型拟合的目标不是 `η(x)`,而是 `e(x)·η(x)`——被倾向分**乘性压低**了。

- 偏差 = `η(x) − e(x)·η(x) = η(x)·(1 − e(x))`,**永远向下**(把风险学低)。
- **哪里最狠?** `e(x)` 小(很少被审)且 `η(x)` 高(其实高危)的地方——正是"**低金额但真的是洗钱**"这批
  被你当成干净负样本的账户。这就是"注入假阴"的精确含义。

```python
import numpy as np, lightgbm as lgb

def fit(X, y, w=None):
    m = lgb.LGBMClassifier(n_estimators=300, learning_rate=0.05,
                           num_leaves=63, random_state=42, verbose=-1)
    m.fit(X, y, sample_weight=w)
    return m

sel = (S == 1)

# ❌ 头号错误:未审 = 0
y_tilde = np.where(S == 1, y, 0)
m_vbias = fit(X, y_tilde)              # 学到的是 e(x)·η(x),不是 η(x)
```

nb13 里这一步把 PR-AUC 从 oracle 的 **0.671** 砸到 **0.542**(−0.13)。

---

## 4. 主缓解:reject inference 为什么对(且不需要花哨算法)

只用**被审过、有真标签**的样本训练,**不给未审的臆造 `y=0`**:

```python
# ✅ reject inference:只用 S==1 的真标签
m_reject = fit(X[sel], y[sel])
```

为什么不偏?看被审子群里的条件概率,再用一次 MAR(`S` 与 `y` 在给定 `x` 下独立):

```
P(y=1 | x, S=1) = P(y=1, S=1 | x) / P(S=1 | x)
               = P(y=1|x)·P(S=1|x) / P(S=1|x)      (MAR: S ⊥ y | x)
               = P(y=1 | x) = η(x)
```

**逐个 `x` 上,你学到的条件概率就是真相 `η(x)`**——没有第 3 节那个 `e(x)` 压低因子。所以"纪律"(不造负标签)
本身就把绝大部分损害拿回来:nb13 里 **0.542 → 0.638**,已很接近天花板 0.671。

> 一句话:**病害来自"臆造标签",不来自"少了点数据"。** 换掉伪标签,病就好了大半——这一步不需要任何高级算法。

---

## 5. 残余的"协变量偏移",以及 IPW 为什么常常只是小精修

reject 之后逐点无偏了,但训练集里 **`x` 的分布**仍是歪的:

```
P(x | S=1) ∝ e(x)·P(x)   ≠   P(x)
```

被审样本**过度代表了"像病人的"区域**(高金额)。这对模型的影响取决于模型:

- **灵活模型(GBDT)** 逐点学 `η(x)`,只要每个区域都**有数据**(overlap),在哪儿都学得对 → reject 已经够好。
- **参数 / 欠拟合模型**(逻辑回归等)必须在各区域间**折中**误差,分布歪了折中点就歪 → 需要纠。

**IPW(逆倾向加权)** 纠这个分布歪:给被审样本乘权重 `1/e(x)`,把稀有区域的样本放大回它在全人群里的份量:

```python
# 先估 e(x)=P(S=1|x)
e = fit(X, S).predict_proba(X)[:, 1]
e = np.clip(e, 0.03, 0.97)            # positivity:别让权重爆炸
m_ipw = fit(X[sel], y[sel], w=1.0 / e[sel])
```

**为什么能还原全人群?** 一个恒等式(Horvitz–Thompson / 重要性加权),`L` 是损失:

```
E_x,y,S [ (S / e(x)) · L(f(x), y) ]
   = E_x,y [ L · E[S | x,y] / e(x) ]
   = E_x,y [ L · e(x) / e(x) ]        (MAR: E[S|x,y]=E[S|x]=e(x))
   = E_x,y [ L ]  = 全人群风险
```

用 `S=1`(有标签)的样本、乘 `1/e(x)`,期望上等于在**整个人群**上算损失。前提两条:**positivity**(`e(x)>0`,否则除零 / 权重爆)
和 **MAR**。

**但对 GBDT + overlap,它就是个小精修**:nb13 里 `reject_ipw − reject ≈ −0.005`(几乎 0、甚至略负,因为权重放大了方差)。
**这不是 bug,是真话**——硬把它调成"IPW 大胜"就是造假(违反本项目如实报告纪律)。IPW 的真正价值在
**参数模型 / 无 overlap / 更强 MNAR** 的场景。

---

## 6. 边界:MAR vs MNAR(面试必须主动讲的地方)

- **MAR(缺失随机、可由 `x` 解释)**:审不审只取决于**你看得见的** `x`(如金额)。上面 reject 无偏、IPW 可纠,**都依赖这条**。
- **MNAR(缺失不可解释)**:审查员用了**你数据里没有的信息**(线人情报、直觉),使得给定 `x` 后 `S` 仍与 `y` 相关:

```
P(y=1 | x, S=1) ≠ P(y=1 | x)
```

这时 reject **也偏了**,而 IPW 用 `e(x)`(只含可见 `x`)**根本纠不动**——偏差的来源不在 `x` 里。**真实 AML 大量是 MNAR。**

---

## 7. 速查表(数字来自 nb13,合成 stand-in,test base rate ≈ 0.17)

| 做法 | 学到的目标 | 偏不偏 | PR-AUC |
|---|---|---|---|
| assume-negative(未审=0)❌ | `e(x)·η(x)` | 向下偏 `η·(1−e)` | 0.542 |
| reject inference(只训已审)✅ | `η(x)`(MAR) | MAR 下逐点无偏 | 0.638 |
| + IPW(`1/e(x)`) | `η(x)` + 纠协变量偏移 | GBDT 下几乎不变 | 0.633 |
| oracle(全真相,合成才有) | `η(x)` | 天花板 | 0.671 |

> 数字是合成实验台的 stand-in、待换 AMLSim 后重估;但**三个方向性结论稳健**:①assume-negative 显著伤害;
> ②reject inference 大幅回收;③IPW 对 GBDT 只是小精修。

## 8. 面试话术(把边界讲成资深信号)

> "我训练用的标签是历史风控策略的产物。**头号偏差是把未审案子当负样本**——我用 **reject inference**(只训被裁定过的)
> 拿回主要部分;在 **MAR** 下还能用 **IPW** 纠协变量偏移,但对树模型收益有限;**在 MNAR 下谁都不能完全纠正**,
> 我不会声称'修好了',只会给出'纠了多少、还剩多少'。"

**知道自己纠不动哪部分**,恰恰比"我全修好了"更可信。
